문제인식

강력한 CAD·BIM은 여전히 로컬에 갇혀 있습니다

대기업이 만든 CAD·BIM 소프트웨어는 고가의 설치형으로, 특정 PC에서만 동작합니다. 협업·접근성·기기 호환 면에서 본질적인 한계가 있습니다.

솔루션 방향

설계 SW도 웹·하이브리드로 전환됩니다

포토샵 → 캔바, 프리미어 → 캡컷처럼 디자인 도구는 웹·경량 설치형으로 이동 중입니다. 빔이랑은 하이브리드 모델로, 데이터는 웹에서 다루되 클라이언트 하드웨어 가속을 활용해 새로운 사용자 경험을 제공합니다.

단계별 제품 진화

  1. 1

    업로드·편집·다운로드 MVP 구현됨

    웹에서 건축 설계 모델을 업로드/다운로드하고, 3D로 가공·편집합니다.

  2. +

    CAD-TO-BIM Add-in MVP 구현됨

    DXF 도면을 객체별(기둥·벽·바닥·지붕) BIM으로 자동 변환합니다.

  3. 2

    생성형 AI 연동

    생성형 API를 연결해 사용자가 원하는 디자인을 생성·수정합니다.

  4. 3

    다중 모델 연결

    여러 모델링을 서로 연결해 더 큰 프로젝트를 구성합니다.
